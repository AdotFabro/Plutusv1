# Plutusv1
Stock analysis dashboard built with Python and Streamlit.

## Features

- Historical stock price charts
- Monthly return analysis
- Company information
- Financial metrics
- Return distributions
- CAPM regression
- Fama-French 3-Factor model

## Data Sources

- Yahoo Finance
- Shiller Historical Data

## Run Locally

pip install -r requirements.txt

streamlit run Plutusv1.py
